JavaScript DOM Manipulation

This project contains a series of JavaScript scripts demonstrating how to interact with and manipulate the Document Object Model (DOM) without reloading the page.
All scripts are designed to run in a Chrome browser (version 57+).

Each task focuses on a specific DOM manipulation technique such as:

Selecting elements with document.querySelector()

Modifying styles and classes

Updating element content

Creating and appending elements

Handling click events

Ensuring the page does not reload during interactions

Following semistandard JavaScript style guidelines

Using let and const instead of var
